从零开始搭建个人博客

环境配置

  1. Windows下安装node.js
  2. Windows下安装cnmp
  3. 安装Hexo

Hexo搭建博客

  1. 初始化:hexo init
  2. 启动项目: hexo s
  3. 访问项目：http://localhost:4000
  4. 编辑内容: MARKDOWN语法: hexo new “index”
  5. 清理项目：hexo clean
  6. 生成项目：hexo g
  7. 启动项目: hexo s
  8. 访问项目：http://localhost:4000

博客部署到Github上

  1. 修改配置文件_config.yml
  2. 项目部署到github：hexo d


  3. 直接在浏览器中访问仓库名
  4. 自定义hexo主题：git clone http://github.com/litten/hexo-theme-yilia themes/yilia
